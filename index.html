ACADEMISEARCH · versiunea 4.0 · actualizat aprilie 2026
1. Descrierea serviciului

AcademiSearch este un motor de căutare academic unificat, gratuit și open-source, destinat studenților, doctoranzilor, cadrelor didactice și cercetătorilor științifici. Aplicația agregează interogări simultan în 42 de surse externe independente și afișează rezultatele pentru a facilita accesul la literatură academică în șapte limbi și surse multilingve.

AcademiSearch este o aplicație client-side (o singură pagină HTML). Nu există server propriu, baze de date proprii sau procesare backend. Toate cererile API sunt efectuate direct din browserul utilizatorului către serviciile terțe listate.

2. Date personale & conformitate GDPR

AcademiSearch nu colectează, nu stochează și nu transmite nicio dată cu caracter personal. Aplicația este conformă cu Regulamentul (UE) 2016/679 (GDPR) prin proiectare (privacy by design).

Specific:

  • Nu există conturi de utilizator, înregistrare sau autentificare
  • Nu se utilizează cookie-uri de tracking, analitics sau publicitate
  • Nu se înregistrează istoricul căutărilor pe niciun server
  • Nu se transmite adresa IP sau User-Agent-ul utilizatorului către operatorul aplicației
  • Cheile API (CORE, Semantic Scholar) introduse de utilizator sunt stocate exclusiv în localStorage al browserului local și nu părăsesc dispozitivul utilizatorului decât în headerele cererilor direct către API-urile respective

Interogările de căutare sunt transmise de browser direct către API-urile serviciilor terțe (arXiv, OpenAlex, CrossRef, Zenodo, DOAJ, Europe PMC, Open Library, Archive.org, Gutenberg, CORE / OpenAIRE, Semantic Scholar). Utilizatorul trebuie să consulte politicile de confidențialitate ale fiecărui serviciu terț accesat.

Excepție — proxy CORS pentru arXiv și Semantic Scholar: în cazul în care browserul blochează cererile directe din cauza politicii CORS, aplicația rutează automat aceste două cereri (doar acestea două) prin serviciul public api.allorigins.win. Acesta acționează ca subprocesor tranzitoriu și poate observa URL-ul interogării, care conține termenul de căutare. Pentru confidențialitate maximă, dezactivați sursele arXiv și Semantic Scholar din panoul „Surse activate" — restul surselor nu trec prin proxy.

Identificator politicos de contact: cererile către API-urile OpenAlex și Unpaywall includ parametrul mailto=academisearch@alexio.tf / email=academisearch@alexio.tf — o convenție standard în ecosistemul academic („polite pool") care permite acestor servicii să contacteze operatorul aplicației în caz de utilizare abuzivă. Acest email aparține operatorului și nu transmite nicio dată a utilizatorului.

3. Cookie-uri și stocare locală

AcademiSearch utilizează exclusiv localStorage al browserului, strict necesar pentru:

  • Salvarea cheii API CORE (opțional, la solicitarea explicită a utilizatorului)
  • Salvarea cheii API Semantic Scholar (opțional, la solicitarea explicită a utilizatorului)

Nu se utilizează cookie-uri de niciun fel. Datele din localStorage pot fi șterse oricând din setările browserului. Conform art. 5 alin. (1) lit. (c) GDPR, stocarea este minimă și strict necesară funcționării opționale solicitate.

4. Proprietate intelectuală și drepturi de autor

AcademiSearch este un instrument de referință și indexare. Aplicația nu stochează, nu reproduce și nu redistribuie conținut protejat de drepturi de autor.

Rezultatele afișate provin din API-uri publice ale surselor terțe. Titlurile, rezumatele și metadatele afișate sunt furnizate de sursele respective sub propriile lor licențe (de ex. Creative Commons, licențe open-access). Utilizatorul este responsabil de respectarea condițiilor de utilizare ale fiecărei surse accesate.

  • Conținutul academic poate fi protejat prin drepturi de autor conform legislației naționale și internaționale
  • Accesul la PDF-uri se face prin link direct către serverele surselor terțe
  • Descărcarea și utilizarea documentelor trebuie efectuată cu respectarea licenței fiecărui document
5. Accesul la literatura academică — instrumente legitime

AcademiSearch utilizează exclusiv instrumente și surse legal disponibile pentru a facilita accesul la literatura academică. Aplicația nu conține, nu promovează și nu face trimitere la nicio platformă care operează în afara cadrului legal al dreptului de autor.

Unpaywall

Unpaywall este o bază de date non-profit care indexează peste 50 de milioane de articole cu versiuni legale open-access, colectate direct de la editori, repozitoarii instituționale și PubMed Central. Acoperă aproximativ 50% din articolele publicate după anul 2000 și este considerat cel mai precis instrument OA disponibil public.

AcademiSearch utilizează API-ul public Unpaywall în două moduri: căutare prin DOI (api.unpaywall.org/v2/{doi}) și căutare prin titlu (api.unpaywall.org/v2/search?query=...). Niciun cont sau autentificare nu este necesară. Limita de utilizare: 100.000 cereri/zi (depășit cu mult de necesitățile acestei aplicații).

OpenAIRE EXPLORE

OpenAIRE EXPLORE este un portal pan-european finanțat de Comisia Europeană prin programul Horizon Europe, care oferă acces la 145M+ publicații, seturi de date și programe software din repozitoarii globale. AcademiSearch integrează API-ul public OpenAIRE (api.openaire.eu/search/publications) ca sursă secundară de verificare OA, după Unpaywall.

Strategia automată de acces în 4 trepte: la apăsarea butonului 🔓 Acces Liber, aplicația: (1) caută DOI via CrossRef dacă lipsește, (2) interoghează Unpaywall prin DOI, (3) interoghează OpenAIRE prin DOI, (4) caută prin titlu în Unpaywall. Dacă nicio versiune OA nu este găsită, utilizatorul este redirecționat spre pagina Unpaywall pentru căutare manuală.

Notă istorică: aplicația a utilizat anterior și API-ul Open Access Button, care a încetat să funcționeze pe 18 noiembrie 2025. Echipa OA.Works a recomandat Unpaywall drept alternativă principală.
